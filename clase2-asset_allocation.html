Construcci√≥n de un portfolio equally weighted#

Un portfolio equally weighted consiste en invertir el mismo porcentaje del dinero asignado en cada uno de los activos del portfolio, esto es, $w_i = 1/n$, donde $n$ corresponde a la cantidad de activos en el portfolio. Para este ejemplo, construiremos un portfolio con 5 acciones: TSLA, IBM, AAPL, NVDA, INTC.

#!pip install yfinance

import numpy as np

n = len(activos)

precios = yf.download(activos, start = "2023-01-01", end = "2023-11-10", auto_adjust=True)

Luego, deberemos definir una lista donde cada componente sea igual a $1/n = 1/5 = 0.2$. Para ello usamos la siguiente sintaxis: n * [1/n]

pesos = n * [1/n]
pesos

[1.0]

Ahora, lo que deberemos hacer es multiplicar la matriz de retornos por el vector de pesos para obtener la rentabilidad del portfolio, esto es:

$$r_{p,t} = \sum_{i=1}^n w_{i,t} \times r_{i,t} = W_t (r_t)‚Äô$$

Entonces, deberemos invertir la matriz de retornos usando el m√©todo T, y haciendo la sumaproducto. Esto √∫ltimo se hace con el m√©todo dot de Numpy. Finalmente, convertiremos el resultado en formato pandas para poder seguir trabajando con los resultados.

rp = pd.Series(np.dot(pesos, retornos.T), index=retornos.index)

Frontera eficiente#

La Frontera eficiente corresponde a las mejores combinaciones de riesgo y retorno de un conjunto de activos. Para conocer esta frontera, se debe resolver el siguiente problema:

$$\min W‚Äô \Sigma W $$ $$s.a. E(r_p) = W \times E(r_i)$$

En otras palabras, la frontera eficiente son las mejores combinaciones de riesgo-retorno que podemos encontrar con $n$ activos. Para construirla debemos resolver este problema de optimizaci√≥n dado los distintos retornos esperados que necesitemos. Una forma sencilla de realizarlo es a trav√©s del m√≥dulo optimizede scipy.

import numpy as np
import scipy.optimize as sco

Dadop que este es un proceso iterativo, deberemos construir algunas funciones:

def get_portf_rtn(w, avg_rtns):
    return np.sum(avg_rtns * w)

def get_portf_vol(w, avg_rtns, cov_mat):
    return np.sqrt(np.dot(w.T, np.dot(cov_mat, w)))

def get_efficient_frontier(avg_rtns, cov_mat, rtns_range):
    
    efficient_portfolios = []
    
    n_assets = len(avg_returns)
    args = (avg_returns, cov_mat)
    bounds = tuple((0,1) for asset in range(n_assets))
    initial_guess = n_assets * [1. / n_assets, ]
    
    for ret in rtns_range:
        constr = (
            {"type": "eq", 
             "fun": lambda x: get_portf_rtn(x, avg_rtns) - ret},
            {"type": "eq", 
             "fun": lambda x: np.sum(x) - 1}
        )
        ef_portf = sco.minimize(get_portf_vol, 
                                initial_guess, 
                                args=args, method="SLSQP", 
                                constraints=constr,
                                bounds=bounds)
        efficient_portfolios.append(ef_portf)
    
    return efficient_portfolios

Para este ejemplo asumiremos que el retorno esperado es igual al retorno promedio hist√≥rico. Este es un p√©simo supuesto, dado que estamos considerando que el pasado es igual al futuro. Dado que tenemos datos diarios, lo anualizaremos multiplic√°ndolos por 252 d√≠as. A su vez, calcularemos la matriz de covarianza de forma an√°loga.

avg_returns = retornos.mean() * 252
cov_mat = retornos.cov() * 252

Definiremos distintos valores objetivo de retorno esperado:

rtns_range = np.linspace(0.1, 0.55, 200)

Luego, correremos el proceso de optimizaci√≥n:

efficient_portfolios = get_efficient_frontier(avg_returns, 
                                              cov_mat, 
                                              rtns_range)

Rescataremos las varianzas del portfolio desde el proceso de optimizaci√≥n:

vols_range = [x["fun"] for x in efficient_portfolios]

Finalmente, graficamos:

import matplotlib.pyplot as plt
import seaborn as sns

with sns.plotting_context("paper"):
    fig, ax = plt.subplots()
    ax.plot(vols_range, rtns_range, "b--", linewidth=3)
    ax.set(xlabel="Volatility", 
           ylabel="Expected Returns", 
           title="Efficient Frontier")

    sns.despine()
    plt.tight_layout()

import pandas_datareader.data as web

# three factors 
factor_3_df = web.DataReader("F-F_Research_Data_Factors", 
                             "famafrench", 
                             start=START_DATE,
                             end=END_DATE)[0]

# momentum factor
momentum_df = web.DataReader("F-F_Momentum_Factor", 
                             "famafrench", 
                             start=START_DATE,
                             end=END_DATE)[0]

# join all datasets on the index
factor_4_df = factor_3_df.join(momentum_df).join(y)

# rename columns
factor_4_df.columns = ["mkt", "smb", "hml", "rf", "mom", "rtn"]

# divide everything (except returns) by 100
factor_4_df.loc[:, factor_4_df.columns != "rtn"] /= 100

# calculate excess returns
factor_4_df["excess_rtn"] = (
    factor_4_df["rtn"] - factor_4_df["rf"]
)
